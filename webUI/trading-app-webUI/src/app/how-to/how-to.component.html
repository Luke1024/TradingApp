How to

  1. Account creation
    • Click "Trading" tab.
    • Click "Add account".
    • Name your account.
    • Use default settings for leverage and balance or set custom values:
      • what is leverage: https://www.investopedia.com/ask/answers/06/forexleverage.asp
      • set account balance in $
    • Click "Save account"
  2. Add order
      3. Click "Add order"
    • Open order using default values or set custom values:
      • When setting take profit and stop loss value backend return tp and sl value in given currency based on lot value.
      • What are take profit and stop loss https://fxscouts.com/forex-education/what-is-stop-loss-and-take-profit/
      • Set buy or sell.
    • Click "Open order"

After the order is opened values are updated based on real-time data every 5 minutes. When the order reach take-profit or stop-loss value orders are automatically closed.

You can create as many accounts and orders per account as you want.
